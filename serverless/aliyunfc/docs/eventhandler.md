事件请求处理程序（Event Handler）

# 基本信息

请求处理程序分为事件请求处理程序（Event Handler）和 HTTP 请求处理程序（HTTP Handler）；其中事件请求由各种事件源触发生成，HTTP 请求则由 HTTP 触发器触发生成。

本文关注 Event Handler。
