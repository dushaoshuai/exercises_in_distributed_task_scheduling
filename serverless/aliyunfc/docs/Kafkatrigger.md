
# 并发配置

| 参数      | 解释                  | yaml 中配置项 | 详细解释                                           | 
|---------|---------------------|-----------|------------------------------------------------|
| 投递并发最大值 | Kafka 投递到函数计算的并发最大值 |           | Kafka 消息投递到函数计算的并发最大值，取值范围为 1~300。该参数仅对同步调用生效。 |
